# v0.5.0-live-data

Release name: Live Data Display MVP  
Release date: 2026-05-14

## Goal

Make the app deployable as a PWA that can show actual KR/US/crypto data from official APIs, public APIs, stored DB rows, or official widgets.

## Included

- `DATA_MODE=live` policy that blocks mock price/rate/chart fallback.
- Provider fetch results with status, HTTP code, raw text snippet, missing env, and error messages.
- `DataProviderStatus` model and `/api/provider-status`.
- `/data-status` UI for provider status.
- `/api/admin/refresh-all` direct job execution.
- Public crypto runtime rendering from Binance, Upbit, and Alternative Fear & Greed.
- Banned production UI copy check for premium/user/FOMO text.

## Out Of Scope

- Premium/ads/payment UI.
- App-internal user behavior rankings or reports.
- Screen scraping from Yahoo, Naver, TradingView, or widgets.
- Play Store/App Store native wrapping.

## Runtime Notes

Local smoke testing on 2026-05-14 succeeded for Binance BTCUSDT, Upbit KRW-BTC, and Alternative Fear & Greed. Database-backed KR/US provider persistence still requires configured env and DB migration in the deployment environment.
