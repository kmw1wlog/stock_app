# v0.5.0-live-data

Release name: Live Data Display MVP  
Release date: 2026-05-14

## Goal

앱을 PWA 또는 스토어 래핑 대상으로 배포했을 때 국장, 미장, 코인 데이터를 실제 API, 공식 공개 조회, 공식 위젯 기준으로 보여준다.

## Scope

- 실제 provider fetch 결과를 구조화한다.
- provider 상태를 DB/API/UI에서 확인 가능하게 만든다.
- refresh-all은 API self-call이 아니라 job 함수를 직접 실행한다.
- live 모드에서 mock 가격, mock 등락률, mock 차트를 노출하지 않는다.
- 최소 하나 이상의 keyless public crypto provider가 실제 데이터를 받아 앱에 렌더링되도록 한다.

## Out of Scope

- 프리미엄/광고/과금 UI
- 앱 내부 사용자 기반 랭킹과 리포트
- Yahoo/Naver/TradingView 화면 크롤링
- Play Store/App Store 네이티브 래핑
