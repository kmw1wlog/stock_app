# v0.4.0-data-mvp

Release name: Data Display MVP  
Release date: 2026-05-13

이번 버전의 목표는 앱 내부 사용자 행동 기반 문구가 아니라 실제 데이터 제공 상태를 정직하게 보여주는 것입니다.

## 구현 방향

- 국장: Data.go.kr EOD 가격, OpenDART 공시, Naver 뉴스 검색 결과를 DB에 저장하는 cron 구조를 구현했습니다.
- 미장: 직접 가격 API가 없으면 자체 등락률을 만들지 않고 TradingView 위젯과 SEC EDGAR 공시 중심으로 표시합니다.
- 코인: Binance/Upbit public API로 24h 가격, 등락률, 거래량, 캔들을 저장하는 구조를 구현했습니다.
- 데이터가 없으면 `데이터 준비중`, `API 키 필요`, `자료 준비중`으로 표시합니다.

## 명확히 비활성화한 것

- 프리미엄 잠금 UI
- 무료 TOP 3 / 프리미엄 전체보기 UI
- 앱 내부 저장률, 조회율, 복사율 기반 문구
- 사용자 행동 기반 랭킹/리포트
- production live 모드의 mock 가격/등락률/차트
